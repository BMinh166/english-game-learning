# \# Vocabulary Learning Roguelike Game

# 

# An educational roguelike card game developed with Godot Engine and GDScript.

# 

# Players build word combinations by identifying linguistic relationships such as synonyms, antonyms, word families, degree relationships, and grammatical categories. The game combines vocabulary learning with strategic decision-making through score optimization, chain mechanics, and item-based progression.

# 

# \## Features

# 

# \- Roguelike progression system

# \- Vocabulary learning through word relationships

# \- Multiple relationship types:

# &#x20; - Synonyms

# &#x20; - Antonyms

# &#x20; - Degree relationships

# &#x20; - Word families

# &#x20; - Word categories

# \- Combo and chain scoring mechanics

# \- Item system that modifies gameplay and scoring strategies

# \- Bag-building style word generation

# \- Persistent progression and save system

# 

# \## Gameplay

# 

# Each round presents a target word.

# 

# Players receive a hand of words and must select one or more words that form valid relationships with the target word.

# 

# Different relationship types award different score values. Chain mechanics, vocabulary difficulty levels, and item effects can further increase the final score.

# 

# After completing a round, players choose support items that influence future gameplay, creating unique builds for each run.
